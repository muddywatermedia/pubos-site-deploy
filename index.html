Architecture
Edge-native. Zero-origin. Sub-50ms.

No servers. No containers. No cold starts. Everything runs at the edge — 300+ locations, automatic failover, infinite scale.

  • ComputeEdge Workers — 8 agents, 300+ edge locations, 0ms cold start
  • DatabaseEdge SQL — SQLite at the edge, automatic replication
  • StorageObject Storage — S3-compatible, zero egress fees
  • CacheEdge KV — Global low-latency key-value store
  • HostingEdge Pages — Dashboard + marketing site
  • AIAI Engine — BYO provider — content generation, social captions
  • EmailEmail Provider — BYO provider — deliverability optimized
  • AuthSigned Auth — HMAC-signed requests, no session tokens to steal
